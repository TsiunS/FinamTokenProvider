Это программа для автоматического получения токена авторизации на сайте https://www.finam.ru.

Токен нужен для скачивания файлов с историческими котировками торговых данных.

## Что делает скрипт
1. Открывает страницу экспорта (например, https://www.finam.ru/quote/moex/sber/export/).
2. Принимает баннер cookie (если есть).
3. Заполняет форму дат/таймфрейма.
4. Нажимает кнопку **«Получить файл»**.
5. Перехватывает URL запроса на скачивание (`export*.out`) и достаёт `finam_token` из query string.

## Почему предыдущая версия часто не получала токен
- Перехват был ориентирован в основном на `Response`, а URL с токеном надёжнее ловить уже на `Request`.
- Заполнение дат шло в ISO (`yyyy-MM-dd`), а форма на странице ожидает `dd.MM.yyyy`.
- Жёсткие селекторы не всегда совпадали с реальной разметкой страницы.

## Результат выполнения
После успешного запуска в рабочей папке появляются:
- `token.txt` — извлечённый токен.
- `debug.log` — сетевой лог с запросами/ответами.
- диагностические скриншоты (`finam-before-download-click.png`, `finam-after-download-click.png`) либо `finam-timeout.png`.

## Запуск
```bash
dotnet restore
dotnet run
```

Если Finam изменит HTML/JS формы, ориентируйтесь на `debug.log` и скриншоты: логика перехвата токена в URL останется рабочей, даже если нужно будет обновить селекторы формы.
